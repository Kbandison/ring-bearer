# Assets

Place your app icons and splash screen images here:

- `icon.png` — 1024×1024 app icon (iOS & Android)
- `adaptive-icon.png` — 1024×1024 foreground layer for Android adaptive icons
- `splash-icon.png` — Splash screen image (recommended 1284×2778 for best results)
- `favicon.png` — 48×48 favicon for web

All images should be PNG format.
